# Ollama + Claude Code Integration

This project supports running Claude Code with open models via Ollama.

## Quick start

```bash
ollama launch claude --model kimi-k2.5:cloud
```

## Headless mode (CI/CD, scripts)

```bash
ollama launch claude --model kimi-k2.5:cloud --yes -- -p "your prompt here"
```

## Recommended models for this codebase
- `kimi-k2.5:cloud` — best for large context (Next.js + Supabase codebase)
- `qwen3.5:cloud` — fast, good for targeted fixes
- `glm-5:cloud` — strong on TypeScript

## Context length
Claude Code needs >= 64k context. Start Ollama with:
```bash
OLLAMA_CONTEXT_LENGTH=64000 ollama serve
```

## Environment (manual setup)
```bash
export ANTHROPIC_AUTH_TOKEN=ollama
export ANTHROPIC_API_KEY=""
export ANTHROPIC_BASE_URL=http://localhost:11434
claude --model kimi-k2.5:cloud
```

## Supabase migrations
To apply pending migrations:
```bash
ollama launch claude --model kimi-k2.5:cloud --yes -- -p "run supabase db push for project xdkojaigrjhzjkqxguxh"
```

Or directly via Supabase CLI:
```bash
export SUPABASE_ACCESS_TOKEN=<your-token>
supabase link --project-ref xdkojaigrjhzjkqxguxh
supabase db push
```
